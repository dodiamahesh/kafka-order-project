💡 What's happening under the hood
1 Angular POSTs order → Spring Boot saves it as PENDING
2 Spring Boot publishes OrderEvent to Kafka topic order-placed
3 @KafkaListener consumer picks up the message from its partition
4 Consumer updates H2 DB status → fires Spring event → SSE pushes to Angular
